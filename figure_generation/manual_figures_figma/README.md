# Manual Figma Figures

Figures 1 and 2 are manually drawn Figma artwork, not Python-generated plots.

Included rendered artwork:
- Figure 1: `rendered_artwork/Figure_1_architecture_contrast.png`
- Figure 2: `rendered_artwork/Figure_2_vgit_pipeline.png`

Included editable vector exports:
- Figure 1: `source/Figure_1.svg`

Native Figma `.fig` files are not required for reproducing the manuscript artwork
and are not redistributed in this package.

Figures 1 and 2 should still be described as manually drawn Figma artwork rather
than Python-reproducible figures.
