# Supplementary Figure S2: Epoch-1 Loss Distribution

Manuscript supplementary figure:
- `rendered_Supplementary_Figure_S2_epoch1_distribution.png`

Run from the repository deposit root:

```bash
python figure_generation/scripted_figures/supp_fig_s2_epoch1/gen_s24_figures.py
```

Script inputs:
- `results_canonical/rq2a_grouping/epoch1_losses.csv`: the 2,880 recorded epoch-1
  training losses. The script reads every value directly; it does not reconstruct or
  sample values from coarse histogram bins.
- `results_canonical/rq2a_grouping/traffic_epoch1_losses.csv` is an included Traffic-only
  projection for audit, but the figure uses the all-run CSV above.

Generated output:
- `figure_generation/scripted_figures/supp_fig_s2_epoch1/figures/s24_epoch1_distribution.png`

Notes:
- Anomaly labels are validated against `final_mse > 0.6` (25 runs, all Traffic).
- The vertical line at epoch-1 loss 0.55 is a recall-first screen, not the anomaly
  definition: TP 25/25 and FP 12/2,855 (0.42% overall; 1.28% among normal Traffic runs).
- The rendered manuscript copy is included as `rendered_Supplementary_Figure_S2_epoch1_distribution.png`.
- Earlier gradient/component side figures not used by the manuscript are intentionally not generated by this public script.
